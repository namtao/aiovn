sWinKey.exe {NUMPADADD} Zoom in using the magnifier. Windows Key + Plus (+)
sWinKey.exe {NUMPADSUB} Zoom out using the magnifier Windows Key + Minus (-)
sWinKey.exe {ESC} Exit magnifier Windows Key + Esc
sWinKey.exe {NUMPADDIV} Start IME reconversion Windows Key + Forward-slash (/)
sWinKey.exe {UP} Maximize app windows Windows Key + Up arrow key
sWinKey.exe {DOWN} Minimize app windows Windows Key + Down arrow key
sWinKey.exe {HOME} Minimize or maximize all but the active desktop window Windows Key + Home
sWinKey.exe {SHIFTDOWN}M{SHIFTUP} Restore minimized windows on the desktop Windows Key + Shift + M
sWinKey.exe {SHIFTDOWN}{UP}{SHIFTUP} Stretch desktop window to the top and bottom of the screen Windows Key + Shift + Up arrow key
sWinKey.exe {SHIFTDOWN}{DOWN}{SHIFTUP} Maximize or minimize active windows vertically while maintaining width Windows Key + Shift + Down arrow key
sWinKey.exe {SHIFTDOWN}{LEFT}{SHIFTUP} Move active window to monitor on the left Windows Key + Shift + Left arrow key
sWinKey.exe {SHIFTDOWN}{RIGHT}{SHIFTUP} Move active window to monitor on the right Windows Key + Shift + Right arrow key
sWinKey.exe {LEFT} Snap app or window left. Windows Key + Left arrow key
sWinKey.exe {RIGHT} Snap app or window right Windows Key + Right arrow key
sWinKey.exe {NUMPAD0}-{NUMPAD9} Open app in number position in the taskbar Windows Key + Number (0-9)
sWinKey.exe {SHIFTDOWN}{NUMPAD0}-{NUMPAD9}{SHIFTUP} Open another instance of the app in number position in the taskbar Windows Key + Shift + Number (0-9)
sWinKey.exe {CTRLDOWN}{NUMPAD0}-{NUMPAD9}{CTRLUP} Switch to last active window of the app in number position in the taskbar Windows Key + Ctrl + Number (0-9)
sWinKey.exe {ALTDOWN}{NUMPAD0}-{NUMPAD9}{ALTUP} Open Jump List of the app in number position in the taskbar Windows Key + Alt + Number (0-9)
sWinKey.exe {CTRLDOWN}{SHIFTDOWN}{NUMPAD0}-{NUMPAD9}{SHIFTUP}{CTRLUP} Open another instance as an administrator of the app in number position in the taskbar Windows Key + Ctrl + Shift + Number (0-9)
sWinKey.exe {CTRLDOWN}{SPACE}{CTRLUP} Change previous selected input option Windows Key + Ctrl + Spacebar
sWinKey.exe {SPACE} Change keyboard layout and input language Windows Key + Spacebar
sWinKey.exe {TAB} Open Task View. Windows Key + Tab
sWinKey.exe {CTRLDOWN}D{CTRLUP} Create a virtual desktop Windows Key + Ctrl + D
sWinKey.exe {CTRLDOWN}{F4}{CTRLUP} Close active virtual desktop Windows Key + Ctrl + F4
sWinKey.exe {CTRLDOWN}{RIGHT}{CTRLUP} Switch to the virtual desktop on the right Windows Key + Ctrl + Right arrow
sWinKey.exe {CTRLDOWN}{LEFT}{CTRLUP} Switch to the virtual desktop on the left Windows Key + Ctrl + Left arrow
sWinKey.exe {CTRLDOWN}{SHIFTDOWN}B{SHIFTUP}{CTRLUP} Wake up the device when black or a blank screen Windows Key + Ctrl + Shift + B
sWinKey.exe {PRINTSCREEN} Capture full screenshot in the "Screenshots" folder Windows Key + PrtScn
sWinKey.exe {SHIFTDOWN}S{SHIFTUP} Create part of the screen screenshot Windows Key + Shift + S
sWinKey.exe {SHIFTDOWN}V{SHIFTUP} Cycle through notifications Windows Key + Shift + V
sWinKey.exe {CTRLDOWN}F{CTRLUP} Open search for device on domain network Windows Key + Ctrl + F
sWinKey.exe {CTRLDOWN}Q{CTRLUP} Open Quick Assist Windows Key + Ctrl + Q
sWinKey.exe {ALTDOWN}D{ALTUP} Open date and time in the taskbar Windows Key + Alt + D
sWinKey.exe . Open emoji panel Windows Key + Period (.) or semicolon (;)
sWinKey.exe {PAUSE} Show System Properties dialog box Windows Key + Pause
